NucleotideReferenceCoordinate

Scope and Usage

ReferenceCoordinate represents a particular contiguous location at which different alleles may be defined with SimpleAllele. NucleotideReferencCoordinate is a ReferenceCoordinate where the underlying reference sequence is a NucleotideReferenceSequence. NucleotideReferenceCoordinate is abstract.

Attributes

None beyond those provided by ReferenceCoordinate.
