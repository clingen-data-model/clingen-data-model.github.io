AminoAcidReferenceCoordinate

  1. Allele Model
  2. Conceptual
  3. ContextualAllele
  4. AminoAcidReferenceCoordinate

AminoAcidReferenceCoordinate defines a location with respect to an AminoAcidReferenceSequence.

Scope and Usage

ReferenceCoordinate represents a particular contiguous location at which different alleles may be defined with ContextualAllele. AminoAcidReferenceCoordinate is a ReferenceCoordinate where the underlying reference sequence is an AminoAcidReferenceSequence

Attributes

None beyond those provided by ReferenceCoordinate.

Related Entities

Relation Name Entity Type Cardinality
referenceSequence AminoAcidReferenceSequence 1

referenceSequence: points to the AminoAcidReferenceSequence on which this AminoAcidReferenceCoordinate is located.
