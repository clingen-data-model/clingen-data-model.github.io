ContextualAllele

An informational artifact that describes an allele as defined on a particular reference sequence.

GENO:0000891

Scope and Usage

ContextualAlleles are reference-dependent identifiers for alleles. They are more completely described in the ClinGen Allele model.

Attributes

Name Type Cardinality Description IRI
relatedCanonicalAllele CanonicalAllele 0..1 SEPIO:0000285
alleleName String 0..1 RDFS:label
legacyNames String 0..* IAO:0000118
produces ContextualAllele 0..* RO:0003000
producedBy ContextualAllele 0..* RO:0003001
