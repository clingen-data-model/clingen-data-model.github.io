CanonicalAllele

One of a set of coexisting sequence variants at a defined genomic locus, or a set of such variants located in cis on the same copy of the chromosome.

GENO:0000890

Scope and Usage

Canonical Alleles are reference-independent identifiers for alleles. They are more completely described in the ClinGen Allele model.

Attributes

Name Type Cardinality Description IRI
preferredCtxAllele ContextualAllele 1..1 SEPIO:0000256
identifier String 1..1 DC:identifier
