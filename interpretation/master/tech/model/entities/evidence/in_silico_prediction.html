InSilicoPrediction

Computationally predicted functional impact of an allele

Descended from Data

Scope and Usage

Computational tools can make predictions about the effect of a sequence allele on the normal function and splicing of a gene product. The InSilicoPrediction element represents these predictions. ContextualAlleles are used because a ContextualAllele represents an allele within a given sequence, e.g. a particular transcript, since in order to make predictions, these tools require a specific transcript to be specified.

Attributes

Name Type Cardinality Description
contextualAllele ContextualAllele 1..1 The contextual allele on which the prediction is made
tool String 0..1 The computational tool or algorithm that makes the prediction
type CodeableConcept 1..1 The type of prediction being made
value String 0..1 The value of the prediction

Inherited Attributes

Name Type Cardinality Inherited From Description
wasGeneratedBy Activity 0..1 Entity The activity that produced this entity
explanation String 0..1 Entity A human-readable description of the entity
