Conservation

Characterizes whether the allele is fixed across species

Descended from Data

Scope and Usage

Conservation represents the degree to which an allele is the same (conserved) across different species. If an allele is the same in many species, it often represents an important functional portion of a gene; variations in the allele may therefore tend to be more pathogenic.

Most conservation calculations are given on a computational score based on a set of multiple sequence alignments across species. To properly describe a conservation data element, it is important to include both the method used and the score that the method provided.

Attributes

Name Type Cardinality Description
allele CanonicalAllele 0..1 The allele for which conservation values are given
method String 0..1 Method or program used to calculate conservation
score float 0..1 Numerical score for the conservation of the allele
conserved boolean 0..1 Whether the allele is conserved

Inherited Attributes

Name Type Cardinality Inherited From Description
wasGeneratedBy Activity 0..1 Entity The activity that produced this entity
explanation String 0..1 Entity A human-readable description of the entity
