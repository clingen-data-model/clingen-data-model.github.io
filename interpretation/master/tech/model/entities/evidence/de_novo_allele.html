DeNovoAllele

Whether or not a particular Allele is DeNovo in an Individual

Descended from Data

Scope and Usage

DeNovo alleles are those alleles in an individual that were not transmitted to them from either of their parents. Deciding that an allele is DeNovo in an individual requires checking for the presence of the allele in the parents. A greater degree of confidence can be achieved by genetic confirmation of the putative parentage.

Attributes

Name Type Cardinality Description
canonicalAllele CanonicalAllele 0..1 The allele whose denovo status is being described
individual Individual 0..1 The individual whose allele may be denovo
maternityConfirmed boolean 0..1 Whether or not maternity was confirmed. False indicates that maternity was not confirmed, not that maternity was found to be incorrect.
paternityConfirmed boolean 0..1 Whether or not paternity was confirmed. False indicates that paternity was not confirmed, not that paternity was found to be incorrect.

Inherited Attributes

Name Type Cardinality Inherited From Description
wasGeneratedBy Activity 0..1 Entity The activity that produced this entity
explanation String 0..1 Entity A human-readable description of the entity
