MendelianDiseaseVariantPathogenicityInterpretation

A statement about the pathogenicity of a variant for a disease

Descended from Data

Scope and Usage

This element represents a pathogenicity interpretation when that interpretation is being used as evidence. Such interpretations may come from an external source, such as ClinVar, or may be an earlier interpretation from an internal source.

Attributes

Name Type Cardinality Description
canonicalAllele CanonicalAllele 0..1 The allele whose pathogenicity is being described
condition Condition 0..* The condition for which the allele's pathogenicty is stated
modeOfInheritance CodeableConcept 0..1 Mode of inheritance for the condition, as defined in HP terms
clinicalSignificance CodeableConcept 0..1 The coded pathogenicity of the allele for the condition

Inherited Attributes

Name Type Cardinality Inherited From Description
wasGeneratedBy Activity 0..1 Entity The activity that produced this entity
explanation String 0..1 Entity A human-readable description of the entity
