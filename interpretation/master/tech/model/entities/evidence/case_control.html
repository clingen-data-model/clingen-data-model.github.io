CaseControl

Allele frequencies as a function of case-control status

Descended from Data

Scope and Usage

CaseControl represents data relating the frequency of a given CanonicalAllele to the case-control status for a given Condition. The frequencies of the allele in these groups is represented by GroupAlleleFrequency elements. Because PopulationAlleleFrequency descends from GroupAlleleFrequency, it may also be used, usually for the control group frequency.

Results of case control statistical analysis are presented as an odds ratio with a confidence level indicating the degree of confidence that the odds ratio is between the given upper and lower bounds.

Attributes

Name Type Cardinality Description
canonicalAllele CanonicalAllele 1..1 The allele studied in the case-control experiment
condition Condition 1..1 The condition studied in the case-control experiment
caseGroupFrequency GroupAlleleFrequency 0..1 The group of individuals with the condition
controlGroupFrequency GroupAlleleFrequency 0..1 The group of individuals without the condition
oddsRatio float 0..1 Ratio of the probability of having the condition given the allele to the probability of the condition without the allele.
confidenceLevel float 0..1 Confidence level in percent
confidenceIntervalLower float 0..1 Lower bound of the confidence interval for the odds ratio
confidenceIntervalUpper float 0..1 Upper bound of the confidence interval for the odds ratio

Inherited Attributes

Name Type Cardinality Inherited From Description
wasGeneratedBy Activity 0..1 Entity The activity that produced this entity
explanation String 0..1 Entity A human-readable description of the entity
