FunctionalData

Data about the function of a gene product

Descended from Data

Scope and Usage

FunctionalData comprises any experimental measurement of the function of a gene, usually (but not always) in relation to the presence of an allele. Many classes of in vivo and in vitro experimental data may be included. Rather than attempt a full classification of the types of functional data and models for each, this model provides a single type of functional data with several optional elements to tag relevant entities such as genes or alleles.

Attributes

Name Type Cardinality Description
contextualAllele ContextualAllele 0..1 Allele defined with respect to the measured transcript
gene Gene 0..1 Gene to which the functional data applies
result String 0..1 Result of the functional experiment
dataType CodeableConcept 0..1 Specific type of functional data collected

Inherited Attributes

Name Type Cardinality Inherited From Description
wasGeneratedBy Activity 0..1 Entity The activity that produced this entity
explanation String 0..1 Entity A human-readable description of the entity
