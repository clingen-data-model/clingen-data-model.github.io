RegionTypesAndAlleles

Whether or not a sequence region is of a certain type or types and/or wether or not sequence region contains a given set of alleles.

Descended from Data

Attributes

Name Type Cardinality Description
referenceSequence ReferenceSequence 0..1 Sequence on which the region is defined
start int 0..1 Start of the region in 0-based coordinates
stop int 0..1 End of the region in 0-based coordinates
isTypes CodeableConcept 0..1
isNotTypes CodeableConcept 0..*
includedAlleles ContextualAllele 0..* Alleles that are within the region
excludedAlleles ContextualAllele 0..* Alleles that are NOT within the region

Inherited Attributes

Name Type Cardinality Inherited From Description
explanation String 0..1 A human-readable description of the entity
wasGeneratedBy Activity 0..1 The activity that produced this entity
wasDerivedFrom Data 0..1 Another entityy which was a former derivation of this entity.
