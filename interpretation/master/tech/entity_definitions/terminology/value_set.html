Please add content

Attributes

Name Type Cardinality Description IRI
description string 0..1 Freetext description of a ValueSet
conceptListExtensibility @id 1..1 The type of valueset is either extendsible or fixed, which determines whether or not the list of concepts defined for the value set can be extended or not when bound to specific attributes. DC:identifier
sourceIdentifierSystems IdentifierSystem 1..* DC:description
concept DomainEntity 0..* An individual entry in a ValueSet

Examples

ID description conceptListExtensibility sourceIdentifierSystems concept
