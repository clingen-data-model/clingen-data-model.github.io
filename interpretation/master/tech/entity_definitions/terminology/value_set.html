A set of identifiers that can be used as the values of specific properties

SEPIO-CG:02000

Please add content

Attributes

Name Type Cardinality Description IRI
label string 0..1 Freetext name of a ValueSet RDFS:label
description string 0..1 Freetext description of a ValueSet DC:description
conceptListExtensibility ValueSetConceptListExtensibility 1..1 The type of valueset is either extendsible or fixed, which determines whether or not the list of concepts defined for the value set can be extended or not when bound to specific attributes.
sourceIdentifierSystems IdentifierSystem 1..*
concept DomainEntity 0..* An individual entry in a ValueSet
conceptType string 1..1
