From SO: A region (or regions) that includes all of the sequence elements necessary to encode a functional transcript. A gene may include regulatory regions, transcribed regions and/or other functional sequence regions.

SO:0000704
Descended from DomainEntity

Scope and Usage

A gene describes a collection of transcripts. Gene identifiers are granted by naming systems, most importantly HGNC.

Attributes

Name Type Cardinality Description IRI
geneSymbol string 1..1 Symbol for a gene provided by a naming system ERO:0000574

Inherited Attributes

Name Type Cardinality Description IRI
description string 0..1 A freetext description of the entity
label string 0..1 A freeform text (preferred) representation used by the authoring system to identify a concept.
userLabel UserLabel 0..1 An optional label defined by the user. Used for custom entities or to clarify the preferred user label on existing entities with non-preferred labels.
