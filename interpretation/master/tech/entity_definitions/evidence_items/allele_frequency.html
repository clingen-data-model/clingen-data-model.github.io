AlleleFrequency

  1. Interpretation Model
  2. Technical Reference
  3. Entity Definitions
  4. Evidence Item

A statement that describes the observed frequency of an allele as determined in a defined group of individuals (typically a representative sampling that serves as a proxy for some larger population of individuals).

SEPIO:0000227
Descended from Statement

Scope and Usage

The AlleleFrequency data element captures data related to a measurement of a particular allele's frequency in a given set of people. It can be used to represent population data, such as that found in ExAC, or it can be used to represent ad hoc groups such as a case or control group.

The attributes of AlleleFrequency provide information on the population background of the group and the purpose for which the group was ascertained, as well as details of the frequency to allow users to understand the quality of the frequency estimate.

Attributes

allele CanonicalAllele 0..1 The variant for which the frequency is given SEPIO:0000275

individualCount integer 0..1 Number of individuals sampled SEPIO:0000302
homozygousAlleleIndividualCount integer 0..1 Number of individuals in the group that are homozygous for the allele SEPIO:0000301
heterozygousAlleleIndividualCount integer 0..1 Number of individuals in the group who are heterozygous for the allele. SEPIO:0000300
hemizygousAlleleIndividualCount integer 0..1 Number of individuals in the group that are hemizygous for the allele SEPIO:0000299
medianCoverage number 0..1 Median coverage at the allele site in the data set SEPIO:0000303
