AlleleMolecularConsequence

  1. Interpretation Model
  2. Technical Reference
  3. Entity Definitions
  4. Evidence Item

A statement that describes the expected effect of a variant allele at the level of genomic or transcript sequence

SEPIO:0000229
Descended from Statement

Scope and Usage:

An allele can be classified by the effect that it has on the gene product. For instance, an allele may create a stop codon or modify an amino acid. These effects are considered as a consequence; terms for these consequences have been compiled in the Sequence Ontology.

Attributes

Name Type Cardinality Description IRI
contextualAllele ContextualAllele 1..1 The contextual (reference-specific) allele that is being classified SEPIO:0000275
