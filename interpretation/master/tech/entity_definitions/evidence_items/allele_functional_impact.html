AlleleFunctionalImpact

  1. Interpretation Model
  2. Technical Reference
  3. Entity Definitions
  4. Evidence Item

A statement describing a particular experimental result about the impact of an allele on gene product function (typically that of the directly affected gene).

SEPIO:0000241
Descended from Statement

Scope and Usage

AlleleFunctionalImpact comprises any experimental measurement of the function of a gene, usually (but not always) in relation to the presence of an allele. Many classes of in vivo and in vitro experimental data may be included. Rather than attempt a full classification of the types of functional data and models for each, this model provides a single type of functional data with several optional elements to tag relevant entities such as genes or alleles.

Attributes

Name Type Cardinality Description IRI
contextualAllele ContextualAllele 0..1 Allele defined with respect to the measured transcript SEPIO:0000275
gene @id 0..1 Gene to which the functional data applies SEPIO:0000278
resultDescription string 0..1 Result of the functional experiment DC:description
assayType @id 0..1 Specific type of functional data collected SEPIO:0000255
