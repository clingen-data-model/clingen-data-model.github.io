RegionAlleles

A statement about whether a region of genomic sequence overlaps one or more particular alleles.

SEPIO:0000244

Descended from Statement

Scope and Usage

Given a SequenceRegion and an Allele, the physical overlap of these entities is implicit. That is, it is derivable whether or not the Allele overlaps or lies within the SequenceRegion. This type allows for an explicit statement of overlap or lack thereof.

Attributes

Name Type Cardinality Description IRI
region SequenceLocation 1..1 Region within which alleles are being measured SEPIO:0000279
allele ContextualAllele 0..* Alleles that are within the region SEPIO:0000275
outcome Coding 1..1 Alleles that are NOT within the region SEPIO:0000197
