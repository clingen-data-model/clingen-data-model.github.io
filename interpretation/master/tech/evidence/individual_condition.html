IndividualCondition

Whether or not an Individual has a Condition

Descended from Information

Scope and Usage

IndividualCondition represents the relationship between a particular Individual and a Condition. The boolean property hasCondition may be either true, indicating that the person has the condition, or false, indicating that the person does not have the condition.

Attributes

Name Type Cardinality Description
individual Individual 0..1 The person being assessed for the condition
condition MendelianCondition 1..* The condition that the person may have
hasCondition boolean 0..1 Whether or not the person has the condition

Inherited Attributes

Name Type Cardinality Inherited From Description
source InformationSource 0..*
explanation String 0..1 A human-readable description of the entity
contribution Contribution 0..* The contributions (agent, time and role) made to this entity
