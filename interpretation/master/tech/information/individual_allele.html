IndividualAllele

Whether or not an Individual has an allele

Descended from Information

Scope and Usage

The IndividualAllele entity represents information about one or two alleles and an individual. Alleles, whether listed as primary or secondary are asserted to be alleles that the individual has. The phase indicates the phase between the two alleles, and as such is only used if the two alleles are both heterozygous, and on the same chromosome. This structure can not be used to represent the statement that an individual has no copies of an allele.

Attributes

Name Type Cardinality Description
individual Individual 0..1 Person whose alleles the data represents
primaryAllele CanonicalAllele 0..1 One allele that the person has
primaryZygosity Coding 0..1 Zygosity of the primary allele in the individual
secondaryAllele CanonicalAllele 0..1 A second allele that the person has
secondaryZygosity Coding 0..1 Zygosity of the secondary allele in the individual
phase Coding 0..1 Phase of the primary and secondary individual

Inherited Attributes

Name Type Cardinality Inherited From Description
source String 0..*
explanation String 0..1 A human-readable description of the entity
contribution Contribution 0..* The contributions (agent, time and role) made to this entity
