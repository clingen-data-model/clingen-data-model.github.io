FamilyHistory

Whether or not a Condition occurs in a given Family

Descended from Information

Scope and Usage

Previous incidence of a condition in a family provides important clues as to the mechanism. For instance, if a family has multiple instances of a condition, then a DeNovo allele is an unlikely mechanism. This data element associates a family with a condition and states whether one or more family members, other than the proband, have the condition.

Attributes

Name Type Cardinality Description
proband Individual 0..1 The person that was the original source of the condition in this family
condition MendelianCondition 0..* A condition that the family may or may not have
familyHasCondition boolean 0..1 Whether or not this family has the stated condition

Inherited Attributes

Name Type Cardinality Inherited From Description
source String 0..*
explanation String 0..1 A human-readable description of the entity
contribution Contribution 0..* The contributions (agent, time and role) made to this entity
