AlleleFrequency

The frequency of an allele in a population or a defined group of individuals

Descended from Information

Scope and Usage

The AlleleFrequency data element captures data related to a measurement of a particular allele's frequency in a given set of people. It can be used to represent population data, such as that found in ExAC, or it can be used to represent ad hoc groups such as a case or control group.

The attributes of AlleleFrequency provide information on the population background of the group and the purpose for which the group was ascertained, as well as details of the frequency to allow users to understand the quality of the frequency estimate.

Attributes

Name Type Cardinality Description
allele CanonicalAllele 0..1 The variant for which the frequency is given
population CodeableConcept 0..1 Ancestry of population or group over which the frequency was measured
ascertainment CodeableConcept 0..1 The criteria applied to create the dataset in which the frequency was measured
alleleFrequency float 0..1 Frequency of the named allele in the population
alleleCount int 0..1 Frequency numerator: number of times the allele was observed
alleleNumber int 0..1 Frequency denominator: number of observations
individualCount int 0..1 Number of individuals sampled
homozygousAlleleIndividualCount int 0..1 Number of individuals in the group that are homozygous for the allele
heterozygousAlleleIndividualCount int 0..1 Number of individuals in the group who are heterozygous for the allele.
hemizygousAlleleIndividualCount int 0..1 Number of individuals in the group that are hemizygous for the allele
medianCoverage float 0..1 Median coverage at the allele site in the data set
