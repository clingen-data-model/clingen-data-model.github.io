FamilySegregationData

Cosegregation of an allele and a condition in a single family

Descended from Information

Scope and Usage

The FamilySegregationData element represents co-segregation of an allele and a condition within a particular family. The segregation can be characterized using a series of aggregate counts, such as the number of individuals that have both the allele and the condition, or those with the allele but not the condition.

In addition, or instead, the specific family details can be provided in pedigree file format. The contents of a pedigree file are stored as a string in the pedigree attribute, and the columns, genotypeValues, and affectedValues provide the metadata required to fully interpret the coding of the pedigree file, for instance, mapping 0/1 genotype codes in the pedigree file to named alleles.

Attributes

Name Type Cardinality Description
canonicalAllele CanonicalAllele 1..1 Allele being tested for cosegregation
condition MendelianCondition 1..1 Condition being tested for cosegregation
family Family 1..1 Family in which segregation is being tested
phenotypePositiveAllelePositive int 0..1 Number of (non-proband) individuals with both the condition and the allele
phenotypePositiveAlleleNegative int 0..1 Number of (non-proband) individuals with the condition and without the allele
phenotypeNegativeAllelePositive int 0..1 Number of (non-proband) individuals without the condition and with the allele
phenotypeNegativeAlleleNegative int 0..1 The number of non-proband family members that have neither the allele nor the phenotype
pedigree String 0..* Detailed information about the family structure, affected status, and genotypes in pedigree/linkage format.
columns String 0..* Column names for the pedigree data element
genotypeValues String 0..* Map describing how genotypes are represented in the pedigree
affectedValues String 0..* Map describing how affected status is represented in the pedigree
anyInconsistentSegregations boolean 0..1 Whether phenotype positive/allele negative or phenotype negative/allele positive individuals are in the family
