InSilicoPrediction

Computationally predicted functional impact of an allele

Descended from Information

Scope and Usage

Computational tools can make predictions about the effect of a sequence allele on the normal function and splicing of a gene product. The InSilicoPrediction element represents these predictions. ContextualAlleles are used because a ContextualAllele represents an allele within a given sequence, e.g. a particular transcript, since in order to make predictions, these tools require a specific transcript to be specified.

Attributes

Name Type Cardinality Description
quantitativePrediction float 0..1
transcript ReferenceSequence 0..1 The contextual allele on which the prediction is made
canonicalAllele CanonicalAllele 1..1 The contextual allele on which the prediction is made
algorithm String 0..1 The computational tool or algorithm that makes the prediction
predictionType CodeableConcept 1..1 The type of prediction being made
categoricalPrediction String 0..1 The value of the prediction
