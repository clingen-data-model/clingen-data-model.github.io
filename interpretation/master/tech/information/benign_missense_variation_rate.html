BenignMissenseVariationRate

The amount of benign missense variation in a particular region

Descended from Information

Scope and Usage

If a gene, or a region within a gene, has a low rate of benign missense alleles, then missense alleles in that gene or region are less likely to be benign. This Data represents the rate at which benign missense alleles occur.

The unit over which the benign missense rate is defined may either be an entire Gene, or a portion of a gene as defined by a ContextualRegion.

Attributes

Name Type Cardinality Description
region Locus 0..1 Sequence on which the region is defined
gene Gene 0..1 Gene that contains the missense variants
value Coding 1..1 The rate at which Benign missense variants occur within the region

Inherited Attributes

Name Type Cardinality Inherited From Description
source String 0..*
explanation String 0..1 A human-readable description of the entity
contribution Contribution 0..* The contributions (agent, time and role) made to this entity
