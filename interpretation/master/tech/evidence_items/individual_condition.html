IndividualCondition

  1. Interpretation Model
  2. Technical Reference and Examples
  3. Evidence Item

A statement about whether a individual is affected by or exhibits a particular condition.

SEPIO:0000238
Descended from Statement

Scope and Usage

IndividualCondition represents the relationship between a particular Individual and a Condition. The boolean property hasCondition may be either true, indicating that the person has the condition, or false, indicating that the person does not have the condition.

Attributes

Name Type Cardinality Description IRI
individual Individual 0..1 The person being assessed for the condition SEPIO:0000280
condition MendelianCondition 1..* The condition that the person may have SEPIO:0000276
individualHasCondition ConditionStatus 0..1 Whether or not the person has the condition SEPIO:0000197

Inherited Attributes

Name Type Cardinality Inherited From Description IRI
qualifier string 0..1 Use “NOT” as the value of this property to assert that the statement is negated. ??
evidenceLine EvidenceLine 0..* supporting evidence SEPIO:0000006
description string 0..1 A human-readable description of the entity DC:description
contribution Contribution 0..* The contributions (agent, time and role) made to this entity SEPIO:0000159
source string 0..* DC:source
