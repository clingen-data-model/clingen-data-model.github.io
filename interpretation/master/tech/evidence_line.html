EvidenceLine

A reification of the underlying supporting data which may be one or more data points or assertions. Typically grouped in a specific manner for use in an owning assertion activity (either Variant Interpretation or CriterionAssessment. All evidence may optionally be given a strength by an agent.

Attributes

Name Type Cardinality Description
explanation String 1..1 A human-readable description of the entity
strength CodeableConcept 0..1
hasInformation Information 0..*
contribution Contribution 0..* The contributions (agent, time and role) made to this entity
