Scope and Usage

Attributes

Name Type Cardinality Description IRI
canonicalAllele CanonicalAllele 1..1 The contextual allele on which the prediction is made SEPIO:0000275
transcript @id 0..1 The transcript on which the prediction is made SEPIO:0000283
predictionType @id 1..1 The type of prediction being made SEPIO:0000255
statementOutcome @id 1..1 The value of the prediction SEPIO:0000197
algorithm string 0..1 The computational tool or algorithm that makes the prediction SEPIO:0000130
